What is bird bingo?

Bird bingo is just a simple bingo game played by some of the employees at Capsher related to our Bird Guesser game we also play in person.

Unfortunately the desk we were playing bingo on had to be cleared for an incoming intern and so I proposed moving this online for a shared space and also for a fun weekend project.

Because this was a personal project made for fun, I'm not planning on putting any money into this. If the app gets spammed or has too much traffic and requires server costs, I'll likely shut it down.

I'm a paranoid and nosy developer, what information is being stored about me?

The source code can be found on GitHub here: https://github.com/austin-owensby/BirdBingo. Feel free to make PRs to it and I'll take a look.

Why do you ask for my name? Ultimately you can lie if this bothers you. The name is just used to show who drew which bird to keep a sense of community.

How long is my name stored and where?

Currently the app is deployed using Render and the data is stored in a PostgreSQL database. Once a new game is started, your name is removed from the database.

If you would like me to clear this data, let me know and I'll work to see what I can do.

Locally, your name is stored in your browser's local storage for convenience and can be manually cleared at any time.
